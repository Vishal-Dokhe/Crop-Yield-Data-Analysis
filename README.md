**# Crop Yield Data Analysis**



This project focuses on analyzing crop yield data to identify trends, patterns, and insights using Python, SQL, and Power BI.



**## Workflow**

1\. Collect raw crop yield data

2\. Clean and preprocess data using Python

3\. Perform analysis using SQL queries

4\. Create interactive dashboard using Power BI

5\. Document findings in the MDD file



**## Tools Used**

\- Python (Pandas, NumPy)

\- SQL

\- Power BI



**## Folder Structure**

\- data/ : Raw and cleaned datasets

\- sql/ : SQL queries

\- dashboard/ : Power BI dashboard and images

\- docs/ : Project documentation (MDD file)



**## Outcome**

The project provides insights into crop production trends, yield variations, and helps in decision-making for better crop management.
